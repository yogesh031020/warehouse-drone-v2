**Resource-Constrained Path Planning and Edge AI on ESP32**

### ?? Project Overview
This project focuses on the development of an autonomous drone designed for indoor warehouse environments. It features a custom **A* Path Planning** algorithm optimized to run on an **ESP32**, combined with **TF-Mini LiDAR** for collision avoidance.

### ??? Hardware Stack
*   **Main Processor:** ESP32 (Dual-Core).
*   **Sensors:** TF-Mini LiDAR (Obstacle Avoidance), Ultrasonic (Altitude Hold).
*   **Navigation:** IR Beacon Precision Docking & A* Pathfinding.
